# P8105 Final Project

## The Pulse of NYC: Weather, Events, and the Subway

A data science project examining how extreme weather conditions and major events influence New York City subway ridership patterns.

### Overview

This project analyzed over 2 million hourly ridership records from 2024â€“2025 to identify underlying commuter behavior trends. The raw dataset comprised approximately 21 million rows, processed into a structured format suitable for statistical modeling and visualization.

### Key Analysis Topics

1. **The Goldilocks Zone** - Optimal weather conditions for ridership
2. **The Event Blast Radius** - Impact of major events on subway usage
3. **Station Personalities** - Behavioral patterns across different stations
4. **Rain Sensitivity** - How precipitation affects commuter decisions
